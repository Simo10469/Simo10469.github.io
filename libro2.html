Capitolo 1: Introduzione agli Algoritmi

Un algoritmo è una sequenza finita di istruzioni che risolvono un problema specifico. Ogni algoritmo deve essere preciso, efficiente e deterministico.

Proprietà degli Algoritmi

  • Finitudine: Un algoritmo deve terminare dopo un numero finito di passi.
  • Definiteness: Ogni passo deve essere ben definito e comprensibile.
  • Efficacia: Ogni operazione deve essere semplice da eseguire e comprensibile per la macchina.
  • Involuzione: Ogni passo deve essere utile per arrivare alla soluzione del problema.

Algoritmi Esempio - Ordinamento (Bubble Sort)

Il **Bubble Sort** è un algoritmo di ordinamento che confronta coppie di elementi adiacenti e li scambia se non sono nell'ordine corretto. Viene ripetuto finché l'intera lista non è ordinata.

            
                procedure BubbleSort(list)
                    n = length(list)
                    for i = 1 to n-1
                        for j = 0 to n-i-1
                            if list[j] > list[j+1]
                                swap(list[j], list[j+1])

In un linguaggio di programmazione come **Python**, il Bubble Sort potrebbe apparire come segue:

            
                def bubble_sort(arr):
                    n = len(arr)
                    for i in range(n):
                        for j in range(0, n-i-1):
                            if arr[j] > arr[j+1]:
                                arr[j], arr[j+1] = arr[j+1], arr[j]
                    return arr

                print(bubble_sort([64, 34, 25, 12, 22, 11, 90]))

Questo codice ordina una lista di numeri utilizzando l'algoritmo di ordinamento Bubble Sort.

Capitolo 2: Programmazione e Linguaggi

La programmazione è il processo di scrittura di algoritmi tramite un linguaggio di programmazione. I linguaggi di programmazione consentono agli sviluppatori di comunicare con i computer in modo che possano eseguire istruzioni specifiche.

Linguaggi di Programmazione

  • Linguaggi di basso livello: Come l'**Assembly**, che sono più vicini al linguaggio macchina. Esempio: Assembly x86 (usato per interagire direttamente con l'hardware).
  • Linguaggi di alto livello: Come **Python**, **Java**, **C++**, che sono più facili da leggere e scrivere, poiché utilizzano una sintassi comprensibile all'uomo.
  • Compilatori vs Interpreti:
    • Compilatori: Traducono il codice sorgente in codice macchina tutto in una volta, producendo un programma eseguibile.
    • Interpreti: Eseguono il codice sorgente riga per riga senza generare un file eseguibile separato.

Esempio di Codice in Python

Di seguito un semplice esempio di programma in **Python** che calcola l'area di un cerchio dato il raggio:

            
                def calcola_area_cerchio(radius):
                    pi = 3.14159
                    return pi * radius * radius

                raggio = 5
                print(f"L'area del cerchio con raggio {raggio} è {calcola_area_cerchio(raggio)}")

Capitolo 3: Strutture Dati

Le strutture dati sono modalità per organizzare e memorizzare i dati in modo che possano essere usati più efficientemente. Ogni struttura ha vantaggi e svantaggi a seconda del contesto di utilizzo.

Tipi di Strutture Dati

  • Array: Una sequenza di elementi dello stesso tipo. Gli array sono efficaci quando si vogliono accedere velocemente agli elementi tramite indice.
  • Liste: Una sequenza di dati che può crescere o diminuire dinamicamente. Le liste sono usate quando si prevede una grande varietà nel numero di elementi.
  • Pile (Stack): Struttura dati basata sul principio **LIFO** (Last In, First Out). L'elemento inserito per ultimo è il primo a essere rimosso.
  • Code (Queue): Struttura dati basata sul principio **FIFO** (First In, First Out). L'elemento inserito per primo è il primo a essere rimosso.
  • Alberi: Strutture dati gerarchiche in cui ogni nodo ha un valore e dei figli. Usato per rappresentare gerarchie o strutture ad albero.

Codice Esempio di Pila (Stack) in Python

            
                class Stack:
                    def __init__(self):
                        self.items = []

                    def is_empty(self):
                        return len(self.items) == 0

                    def push(self, item):
                        self.items.append(item)

                    def pop(self):
                        if not self.is_empty():
                            return self.items.pop()

                # Creazione di una pila e operazioni
                stack = Stack()
                stack.push(10)
                stack.push(20)
                print(f"Elemento rimosso: {stack.pop()}")  # Stampa 20

Capitolo 4: Algoritmi e Programmazione

In questo capitolo esploreremo come visualizzare algoritmi complessi tramite diagrammi di flusso (Flowcharts) e come implementare gli algoritmi in linguaggi di programmazione.

Flowchart: Diagramma di Flusso

I flowchart sono diagrammi che rappresentano graficamente un algoritmo. Ogni passo è rappresentato da un simbolo diverso (es. rettangolo per operazioni, rombo per decisioni).

Esempio di Flowchart - Determinare se un numero è pari o dispari

In un flowchart, il processo per determinare se un numero è pari o dispari può essere rappresentato come segue:

Codice Esempio in Python

            
                def verifica_pari_dispari(numero):
                    if numero % 2 == 0:
                        return "Pari"
                    else:
                        return "Dispari"

                print(verifica_pari_dispari(4))  # Stampa "Pari"

Capitolo 5: Le Reti

Le reti informatiche sono sistemi di computer e dispositivi connessi tra loro per scambiarsi dati e risorse. Esistono diversi tipi di reti, ciascuna con caratteristiche uniche in base alla sua estensione e utilizzo.

PAN (Personal Area Network)

Una **PAN** è una rete che connette dispositivi in un'area ristretta, come il tuo smartphone, il computer e una stampante tramite Bluetooth o Wi-Fi.

LAN (Local Area Network)

Una **LAN** è una rete che copre un'area geografica limitata, come una casa o un ufficio. Essa connette computer e altri dispositivi tramite cavi Ethernet o Wi-Fi.

WAN (Wide Area Network)

Una **WAN** è una rete che copre una vasta area geografica, come una città, una nazione o il mondo intero. **Internet** è un esempio di WAN.

Internet

**Internet** è una rete globale che collega milioni di computer e dispositivi, permettendo di scambiare informazioni, utilizzare applicazioni web e accedere a risorse remote.
