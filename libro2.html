Capitolo 1: Introduzione agli Algoritmi

Un algoritmo è una sequenza finita di istruzioni che risolvono un problema specifico. Ogni algoritmo deve essere preciso, efficiente e deterministico.

Proprietà degli Algoritmi

  • Finitudine: Un algoritmo deve terminare dopo un numero finito di passi.
  • Definiteness: Ogni passo deve essere ben definito e comprensibile.
  • Efficacia: Ogni operazione deve essere semplice da eseguire e comprensibile per la macchina.
  • Involuzione: Ogni passo deve essere utile per arrivare alla soluzione del problema.

Algoritmo di Ordinamento (Bubble Sort)

Il **Bubble Sort** è un algoritmo di ordinamento che confronta coppie di elementi adiacenti e li scambia se non sono nell'ordine corretto. Questo processo viene ripetuto finché l'intera lista non è ordinata.

            
                procedure BubbleSort(list)
                    n = length(list)
                    for i = 1 to n-1
                        for j = 0 to n-i-1
                            if list[j] > list[j+1]
                                swap(list[j], list[j+1])

Capitolo 2: Programmazione e Linguaggi

La programmazione è il processo di scrittura di algoritmi tramite un linguaggio di programmazione. I linguaggi di programmazione consentono agli sviluppatori di comunicare con i computer per eseguire istruzioni specifiche.

Linguaggi di Programmazione

  • Linguaggi di basso livello: Come l'**Assembly**, che sono più vicini al linguaggio macchina. Esempio: **Assembly x86**.
  • Linguaggi di alto livello: Come **Python**, **Java**, **C++**, che sono più facili da leggere e scrivere, utilizzando una sintassi comprensibile per l'uomo.
  • Compilatori vs Interpreti:
    • Compilatori: Traducono il codice sorgente in codice macchina prima dell'esecuzione.
    • Interpreti: Eseguono il codice sorgente riga per riga senza produrre un file eseguibile separato.

Esempio di Codice in Python

            
                def calcola_area_cerchio(radius):
                    pi = 3.14159
                    return pi * radius * radius

                raggio = 5
                print(f"L'area del cerchio con raggio {raggio} è {calcola_area_cerchio(raggio)}")

Capitolo 3: Strutture Dati

Le strutture dati sono metodi per organizzare e memorizzare i dati in modo che possano essere utilizzati più facilmente ed efficientemente.

Tipi di Strutture Dati

  • Array: Una sequenza di elementi dello stesso tipo. Utilizzato per l'accesso rapido tramite indice.
  • Liste: Una sequenza di dati che può crescere o diminuire dinamicamente. Utilizzata quando il numero di elementi è variabile.
  • Pile (Stack): Struttura dati che segue il principio **LIFO** (Last In, First Out).
  • Code (Queue): Struttura dati che segue il principio **FIFO** (First In, First Out).

Codice Esempio di Pila (Stack) in Python

            
                class Stack:
                    def __init__(self):
                        self.items = []

                    def is_empty(self):
                        return len(self.items) == 0

                    def push(self, item):
                        self.items.append(item)

                    def pop(self):
                        if not self.is_empty():
                            return self.items.pop()

                # Esempio di utilizzo della Pila
                stack = Stack()
                stack.push(10)
                stack.push(20)
                print(f"Elemento rimosso: {stack.pop()}")  # Stampa 20

Capitolo 4: Algoritmi e Programmazione

In questo capitolo esploreremo come rappresentare e implementare gli algoritmi, compreso l'uso dei diagrammi di flusso (Flowcharts) per visualizzare il processo di un algoritmo.

Flowchart: Diagramma di Flusso

Il diagramma di flusso rappresenta graficamente i passaggi di un algoritmo, dove ogni passo è rappresentato da simboli distinti (ad esempio, un rombo per le decisioni).

Esempio di Flowchart

Ecco un esempio di un diagramma di flusso per determinare se un numero è pari o dispari:

Codice Esempio in Python

            
                def verifica_pari_dispari(numero):
                    if numero % 2 == 0:
                        return "Pari"
                    else:
                        return "Dispari"

                print(verifica_pari_dispari(4))  # Stampa "Pari"

Capitolo 5: Le Reti

Le reti informatiche permettono di connettere computer e dispositivi per scambiare dati e risorse. Esistono diversi tipi di reti, ciascuna con caratteristiche uniche.

PAN (Personal Area Network)

Una **PAN** è una rete di piccole dimensioni che connette dispositivi come smartphone e computer all'interno di un'area limitata (ad esempio, in una stanza).

LAN (Local Area Network)

Una **LAN** è una rete che copre un'area geografica limitata, come una casa o un ufficio. Connetti dispositivi tramite Wi-Fi o cavi Ethernet.

WAN (Wide Area Network)

Una **WAN** è una rete che copre aree geografiche più grandi, come città, paesi o il mondo intero. Internet è un esempio di WAN.

Internet

**Internet** è una rete globale che collega milioni di dispositivi, consentendo la comunicazione e l'accesso a risorse remote.
